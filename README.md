# Resume Lab - 专业简历优化工具

一个专为云原生、SRE、MLOps等专业技术岗位设计的简历优化工具，旨在提升简历与目标职位的匹配度和专业性。

## 项目概述

本项目针对云计算、站点可靠性工程（SRE）、机器学习运维（MLOps）等领域的工程师和技术人员，解决通用简历工具无法精准匹配技术岗位术语和技能要求的问题，提供可解释的匹配分析与个性化优化建议。

## 功能特性

### 🎯 职位匹配分析
- 基于技术栈、角色、场景三维度进行智能评分与命中率报告
- 提供详细的命中/缺失分析，按技术、角色、场景分组展示
- 实时反馈修改对匹配度的影响

### 💡 Bullets生成器
- 提供领域定制化模板（Cloud/SRE/MLOps），支持量化表达和一键复制
- 基于结构化JSON模板驱动内容生成，确保输出风格差异化
- 生成内容自动注入用户勾选的技术标签，保证个性化输出

### 📤 简历导出
- 支持PDF和DOCX格式导出，具备美观的专业模板
- 正确格式化标题、段落、列表，保留加粗等强调样式

### 📊 匿名统计看板
- 管理员可查看用户行为趋势与技能分布，数据完全匿名化
- 实现7天/30天时间范围切换功能
- 具备响应式布局的图表展示

### 🔐 隐私保护
- 浏览器本地存储为主，保障隐私安全
- 不上传用户简历原文，仅允许上传匿名统计特征
- 提供"清除所有项目数据"功能

## 技术架构

### 技术栈
- **前端**: React 18 + Next.js 14.2.35 + TypeScript 5.3.3
- **UI框架**: Tailwind CSS 3.4.1 + Radix UI + Lucide-react
- **图表库**: Recharts 2.12.0
- **文档生成**: docx 8.5.0（用于 DOCX 导出）
- **后端接口**: Next.js API Routes
- **数据库**: PostgreSQL（通过 pg 8.11.3 驱动）
- **状态与类型**: Zod 3.22.4（Schema 验证）、UUID 9.0.1（ID 生成）
- **测试**: Jest 29.x + Testing Library

## 安装与运行

### 环境要求

```bash

# 或
pnpm install
```

```bash

```


### 运行测试
```bash
npm run test          # 运行所有测试
npm run test:watch    # 监听模式运行测试
npm run test:coverage # 生成测试覆盖率报告
```

## 隐私政策

本项目高度重视用户隐私保护：

1. **不上传原始内容**：所有简历分析均在浏览器本地完成，不上传任何原始简历或JD文本
2. **匿名统计**：仅在用户主动开启统计开关后，才上传匿名统计特征（技能标签、学历、年限桶、匹配分、缺失项等）
3. **本地存储**：用户数据优先存储在浏览器本地
4. **数据清除**：提供"清除所有项目数据"功能，可删除所有项目相关数据
